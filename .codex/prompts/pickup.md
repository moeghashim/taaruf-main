# /pickup

Purpose: rehydrate context quickly before doing implementation work.

Checklist:
1. Read `AGENTS.md` and any relevant docs under `docs/`.
2. If the task touches React, Next.js, UI review, or transitions, read `docs/agent-skills.md` and the relevant vendored skill under `agent/skills/vercel-labs/`.
3. Review recent entries in `progress.md` and summarize the key learnings that apply to this task.
4. Run `git status -sb` and summarize working tree state.
5. Identify current branch and any related release or deploy context.
6. Run or review the latest `npm run check`, `npm test`, and `npm run agent:check` results.
7. List the next 2-3 concrete actions before editing.
